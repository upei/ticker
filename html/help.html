Keyboard shortcuts

Note: Windows users should use CTRL instead of CMD

keystroke command
CMD+B Markdown: bold
CMD+I Markdown: italic
CMD+U Markdown: link»
CMD+R Markdown: code
CMD+F toggle between showing all tweets and only @replies & dms
CMD+SHIFT+M Mark all messages as read in current view
↑ or K or CMD+K Up one message
↓ or J or CMD+J Down one message
CMD+↑ Up one reply
CMD+↓ Down one reply
CMD+SHIFT+↑ Up one dm
CMD+SHIFT+↓ Down one dm
CMD+HOME Select first message
CMD+END Select last message
F5 Reload current view
SHIFT+F5 Clear and reload current view
CMD+SHIFT+A Open shorten link window
CMD+T Jump to message entry box
CMD+SHIFT+2 Respond to sender of currently selected message
CMD+, View Preferences
CMD+SHIFT+, Open preferences folder
CMD+{1-7} Switch to Tab 1-7
